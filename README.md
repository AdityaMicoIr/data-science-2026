## Kesimpulan Umum Pembelajaran (Pertemuan 1–7)

Rangkaian pembelajaran dari pertemuan 1 hingga 7 memberikan pemahaman mendasar mengenai alur kerja *Data Science* berdasarkan kerangka **CRISP-DM**. Proses pembelajaran dimulai dari pengenalan konsep dasar, penggunaan Python sebagai alat analisis, serta pemanfaatan library seperti NumPy dan Pandas untuk pengolahan data.

Tahap berikutnya berfokus pada analisis data melalui statistik deskriptif dan visualisasi univariat maupun bivariat untuk memahami pola, distribusi, serta hubungan antarvariabel.

Selain itu, proses *data cleaning* dan *preprocessing* menjadi bagian penting dalam memastikan kualitas data sebelum digunakan untuk analisis lebih lanjut. Penanganan *missing values*, data duplikat, dan *outlier* menunjukkan bahwa kualitas data sangat memengaruhi hasil analisis dan performa model, sejalan dengan prinsip **Garbage In, Garbage Out**.

Pembelajaran ditutup dengan pengenalan konsep *machine learning* melalui implementasi model regresi linear. Pada tahap ini, ditekankan pula pentingnya proses pembagian data (*train-test split*) dan pencegahan *data leakage* agar model dapat dievaluasi secara objektif.

Secara keseluruhan, tujuh pertemuan ini telah membangun fondasi yang kuat, baik dari sisi teknis maupun pola pikir analitis, sebagai bekal untuk mempelajari metode *Data Science* yang lebih lanjut.
